# Images Required

This directory should contain the following image files:

## Main Images (in restaurant/static/img/)

1. **logo.png** - Little Lemon header logo (200x100px)
2. **logo_footer.png** - Little Lemon footer logo (100x50px)
3. **restaurant.jpg** - Restaurant interior/hero image (1200x600px)
4. **grill.jpg** - Grilled food on skewers (800x600px)
5. **salad.jpg** - Fresh salad image (800x600px)
6. **chef.jpg** - Chef cooking in kitchen (800x600px)

## Menu Item Images (in restaurant/static/img/menu_items/)

1. **Bruschetta.jpg** - Photo of Bruschetta (800x600px)
2. **Greek salad.jpg** - Photo of Greek salad (800x600px)
3. **Grilled fish.jpg** - Photo of Grilled fish (800x600px)
4. **Lemon dessert.jpg** - Photo of Lemon dessert (800x600px)

## Where to Get Images

### Option 1: Free Stock Photos

- **Unsplash** (https://unsplash.com/) - Search for: "restaurant", "mediterranean food", "lemon logo", "chef", "salad", "grilled fish"
- **Pexels** (https://www.pexels.com/) - High quality free images
- **Pixabay** (https://pixabay.com/) - Free images and vectors

### Option 2: Create Logo

- **Canva** (https://www.canva.com/) - Free logo maker with lemon icons
- **LogoMakr** (https://logomakr.com/) - Simple online logo creator

### Option 3: Course Materials

If images were provided in your course materials, use those files.

## Important Notes

⚠️ **The website will still function without images**, but image links will be broken.

✅ **To make the website fully functional:**

1. Download or create the images listed above
2. Save them with the exact filenames shown
3. Place them in the correct directories
4. Restart the Django development server
5. Refresh your browser

## Quick Setup

After downloading images:

```powershell
# Navigate to the project directory
cd "d:\Code\coursera_learning\Meta-backend-developer\Django Web Framework"

# Run the server
python manage.py runserver
```

Then visit http://127.0.0.1:8000/ to see your website with images!
